# 안녕하세요

백엔드와 데이터 중심 웹서비스 개발에 관심이 많은 개발자입니다.  
사용자의 행동 데이터를 분석해 더 나은 경험을 설계하는 데 집중하고 있습니다.  
현재는 Spring Boot, JSP, Oracle DB를 기반으로 한 **레시피 추천 플랫폼**과  
**PC방 시스템** 같은 프로젝트를 개발하고 있습니다.
